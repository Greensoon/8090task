# Library Management System - Task 1

## Overview

This is a complete library management system that allows users to:
- Add books and users to the system
- Borrow and return books
- View library datas
- Check user information
- Browse the entire book collection

## 📁 File Structure

```
task1/
├── README.md           # This file
├── main.py             # main entry
├── library_system.py   # library system
├── entities.py         # entity classes
└── abstract_classes.py # Abstract base classes
```

## 📊 Features

- **User Management**: Add students and librarians
- **Book Management**: Add, borrow, and return books
- **Inventory Tracking**: Monitor available copies
- **User Statistics**: Track who has borrowed what
- **Library Analytics**: View collection and user statistics
